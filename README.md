## üöÄ Getting Started
### For Users (Try It Out!)
1. **Clone the Repository:** `git clone https://github.com/zaimhalili/Equathora.git`
2. **Navigate to Frontend:** `cd equathora`
3. **Install Dependencies:** `npm install`
4. **Run Locally:** `npm run dev` ‚Äî open your browser to `http://localhost:5173` and start practicing!
5. **Optional Backend:** For full features, run `cd backend/EquathoraBackend && dotnet run` (requires .NET SDK).

**Need a Hosted Demo?** If you have a live URL, let me know, and I'll add a direct link here!

### For Developers
- **Prerequisites:** Node.js 18+, .NET SDK 9.0+ (for backend).
- **Environment Setup:** Create `equathora/.env.local` with your Supabase keys (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`).
- **Build & Deploy:** Use `npm run build` for production. Deploy frontend to Vercel or similar; backend to Azure App Service.
- **Configuration:** Backend settings in `equathora/backend/EquathoraBackend/appsettings*.json`.
